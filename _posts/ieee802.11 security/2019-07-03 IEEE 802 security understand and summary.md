
# IEEE 802 security 理解与总结
---
## *1. 802 security 简介*

&emsp;&emsp;IEEE 802 安全算法标准定义了两种子类：
* 创建或使用一个RSNA的算法，名为 RSNA 算法；
* Pre-RSNA 算法；

&emsp;&emsp;WEP算法就 机密性、身份验证、访问控制 功能而言，已不再匹配该标准，所以已经遗弃。

&emsp;&emsp;TKIP算法同样也已经遗弃。
